  1. a dictionary (publication that explains the meanings of an ordered list of words)
  2. a letter of resignation
  3. a formal document of abdication when one emperor steps down and a new emperor ascends the throne

What is kusojisho?

kusojisho is a one-page, searchable, offline-ready, static glossary template for Jekyll.

one-page

A web page where all the content is on a single page instead of being spread across multiple pages.

searchable

Capable of being searched, especially something that has an automated search function included.

offline-ready

A web page which can work offline.

static

A website that does not make use of server-side scripting or coding to display information, and thus has no interactivity beyond the user's choice of hyperlink to follow.

Jekyll

A static-site generator written in Ruby.

Features

printable

Thanks to GutenbergCSS, kusojisho glossaries look beautiful when printed.

easy to use

All you really need is a GitHub/GitLab account.

git

kusojisho uses git, which makes it simple for multiple people to work on the same glossary, and makes rollbacks possible if you ever need them.

themable

kusojisho uses sakuracss, which makes it simple to retheme your website. You can also use your own custom CSS.

JS-optional

This website will render properly regardless of you having JS enabled. Some functionaliy, mainly search, will not be available, but you can still use your browser's built-in search via Ctrl+F.
